# 📊 Power BI Sales Analytics Dashboard

## 🚀 Project Overview
This project presents an interactive **Sales Analytics Dashboard** built using Power BI to analyze business performance and generate actionable insights.

The dashboard transforms raw sales data into meaningful visualizations to support business decision-making.

---

## 🎯 Objectives
- Analyze sales performance
- Identify profitable regions & categories
- Track monthly sales trends
- Highlight key business insights

---

## 📈 Dashboard Features
✅ Executive Dashboard  
✅ KPI Cards (Sales, Profit, Orders)  
✅ Sales Trend Analysis  
✅ Regional Performance Map  
✅ Category-wise Sales Distribution  
✅ Insights Page for Business Decisions
